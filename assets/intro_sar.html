🛰️ Módulo SAR - AGLgis

O módulo SAR do AGLgis fornece uma interface gráfica para processar dados Sentinel-1 usando o Sentinel-1 SAR Backscatter Analysis Ready Data no Google Earth Engine, sem necessidade de programação.

📋 Fluxo de Trabalho

  1. Inputs: Selecione a área (AOI), datas e parâmetros de processamento
  2. Run: Execute o processamento para gerar a série temporal
  3. Results: Visualize gráficos, filtre datas, e exporte os resultados

✨ Principais Funcionalidades

  • Seleção de Área e Datas: Interface intuitiva para definir AOI e período de análise
  • Parâmetros de Processamento: Correção de ruído de borda, nivelamento de terreno, filtragem de speckle
  • Múltiplos Índices Espectrais: Escolha entre VV/VH Ratio, RVI ou DpRVI para análise customizada
  • Modos de Renderização Flexíveis: Compostas RGB com contraste aprimorado ou pseudocolor de banda única com paleta Viridis
  • Série Temporal Interativa: Gráfico Plotly com zoom, pan e hover
  • Filtro de Datas: Selecione datas específicas para refinar análise
  • Visualização de Imagens: Pré-visualize ou baixe imagens SAR de cada data
  • Download em Lote: Baixe múltiplas imagens com progresso e cancelamento
  • Exportação CSV: Exporte dados da série temporal com filtro de datas
  • Carregamento Automático: Imagens baixadas são carregadas no QGIS automaticamente

📊 Série Temporal

O plugin gera gráficos interativos mostrando a série temporal da razão VV/VH (VV/VH Ratio) para sua área de interesse. Use o filtro de datas para focar em períodos específicos e exportar os dados para análise externa.

🖼️ Bandas e Índices Espectrais

Cada imagem SAR contém 3 bandas: as duas polarizações (VV, VH) mais um índice espectral selecionável.

  • Band 1 - VV: Polarização vertical-vertical
  • Band 2 - VH: Polarização vertical-horizontal
  • Band 3 - Índice Espectral: Escolha entre:
    • VV/VH Ratio: Razão direta entre as polarizações (padrão)
    • RVI: Radar Vegetation Index: 4 × VH / (VV + VH)
    • DpRVI: Dual-pol SAR Vegetation Index: VH / (VH + VV)

Selecione o índice desejado na aba "Inputs" antes de executar o processamento. O índice escolhido será incluído na série temporal e nas imagens baixadas.

🔧 Configuração Inicial

Para usar este módulo, você precisa de autenticação no Google Earth Engine através de um Google Cloud Project ID. Configure isso na guia "Auth" do plugin.

🛰️ SAR Module - AGLgis

The AGLgis SAR module provides a graphical interface to process Sentinel-1 data using the Sentinel-1 SAR Backscatter Analysis Ready Data in Google Earth Engine, without requiring any coding.

📋 Workflow

  1. Inputs: Select area (AOI), dates, and processing parameters
  2. Run: Execute processing to generate time series
  3. Results: View charts, filter dates, and export results

✨ Main Features

  • Area & Date Selection: Intuitive interface for defining AOI and analysis period
  • Processing Parameters: Border noise correction, terrain flattening, speckle filtering
  • Multiple Spectral Indices: Choose between VV/VH Ratio, RVI, or DpRVI for customized analysis
  • Flexible Render Modes: RGB composites with enhanced contrast or single-band pseudocolor with Viridis palette
  • Interactive Time Series: Plotly chart with zoom, pan, and hover
  • Date Filter: Select specific dates to refine analysis
  • Image Preview: Preview or download SAR images for each date
  • Batch Download: Download multiple images with progress and cancel option
  • CSV Export: Export time series data with date filtering
  • Auto-Load Images: Downloaded images automatically load in QGIS

📊 Time Series

The plugin generates interactive charts showing the VV/VH Ratio (VV/VH Ratio) time series for your area of interest. Use the date filter to focus on specific periods and export data for external analysis.

🖼️ Bands & Spectral Indices

Each SAR image contains 3 bands: the two polarizations (VV, VH) plus a selectable spectral index.

  • Band 1 - VV: Vertical-vertical polarization
  • Band 2 - VH: Vertical-horizontal polarization
  • Band 3 - Spectral Index: Choose from:
    • VV/VH Ratio: Direct ratio between polarizations (default)
    • RVI: Radar Vegetation Index: 4 × VH / (VV + VH)
    • DpRVI: Dual-pol SAR Vegetation Index: VH / (VH + VV)

Select the desired index in the "Inputs" tab before running the processing. The chosen index will be included in the time series and in downloaded images.

🔧 Initial Setup

To use this module, you need authentication to Google Earth Engine via a Google Cloud Project ID. Configure this in the "Auth" tab of the plugin.
